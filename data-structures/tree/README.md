## Binary Tree Js

#Binary search trees for ES6

Create instance of class

```
let tree = new BinaryTree();
```

Use method to manipulate the tree
```
tree.add(value);
tree.search(tree, value)
tree.inorder()
tree.preorder()
tree.postorder()
tree.getParent(value)
tree.getMinimumTree()
tree.getMaximumTree()
```
